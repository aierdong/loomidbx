# Product Overview

LoomiDBX 是一个桌面端数据库模拟/合成数据生成工具，帮助用户快速生成符合真实世界规律的测试数据。它连接用户数据库，识别 Schema，允许为每张表和字段定制生成策略，最终高速生成并回写数据。

## Core Capabilities

- **智能 Schema 识别**：自动扫描数据库表结构、字段类型、外键关系和约束
- **多样化数据生成**：支持序列、概率分布、正则表达式、枚举、外部数据源、计算字段、AI 生成等多种策略
- **关系与约束保证**：自动处理物理/逻辑外键关系（1:1、1:N、M:N），支持唯一性、Null 值比例等约束
- **高性能批量写入**：目标 ≥10,000 条/秒，支持并发生成和事务批量提交
- **配置持久化与复用**：保存连接、Schema 和生成规则，支持 Schema 变更后重新映射

## Target Use Cases

- **产品演示**：市场/售前人员快速生成"漂亮"的 Demo 数据
- **性能测试**：数据库开发/测试人员进行压力测试和性能调优
- **算法研究**：数据分析人员构造符合真实分布的训练数据
- **开发环境填充**：开发人员快速准备开发所需的测试数据

## Value Proposition

**解决痛点**：
- 现有工具生成的数据"太假"（随机字符拼接），无法满足演示需求
- 不支持外键关系和约束，生成的数据无法用于实际开发/测试
- 手动准备数据成本高昂

**核心价值**：
- 生成"逼真"数据：姓名符合百家姓规则、薪水满足帕累托分布、订单编号遵循业务规则
- 满足关系约束：自动处理外键依赖、拓扑排序生成顺序、支持中间表
- 简单易用：借鉴 Navicat/DBeaver 界面风格，降低认知障碍
